# วิธีการรัน Project
**ติดตั้ง package**
```yarn```

**เข้าโฟล์เดอร์ backend**
```cd /backend```

**run backend**
```yarn start:dev```

**ออกจาก backend**
```cd ..```

**run frontend**
```npx expo start```
