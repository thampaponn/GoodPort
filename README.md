# วิธีการรัน Project
** ติดตั้ง package Frontend**

    npm install

**run frontend**

    npx expo start
    

## new Terminal

**เข้าโฟล์เดอร์ backend เพื่อรัน backend**

    cd /backend
    
**ติดตั้ง package**

    yarn


**run backend**

    yarn start:dev
